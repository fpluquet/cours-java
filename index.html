Bienvenue sur le site du cours de Java ​

Ce site regroupe tout le contenu du cours de Java, JavaFX, l'architecture MVC et les Design Patterns pour l'année académique 2024-2025 à la HELHa.

Accès rapide aux chapitres ​

  • Java
  • JavaFX
  • Architecture MVC
  • Design Patterns

À propos ​

Ce syllabus a été conçu par Frédéric Pluquet (HELHa, Année académique 2024-2025), avec une pensée reconnaissante pour Dudziak Thomas, ancien étudiant, qui a contribué à la base de ce document. Ce cours est vivant : il est disponible sur GitLab et toutes suggestions d’amélioration sont les bienvenues.

Technologie pour ce support ​

Ce site est généré avec VitePress, un générateur de site statique basé sur Vue.js, et utilise Markdown pour la rédaction des contenus. Il est publié en open source sous licence MIT, ce qui signifie que vous pouvez l'utiliser, le modifier et le redistribuer librement.

Ce site est hébergé sur GitHub Pages.

Bonne lecture et bon apprentissage !
